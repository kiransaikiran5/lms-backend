# LMS Backend System

A complete Learning Management System backend with JWT authentication, role-based access control, course management, enrollment tracking, progress monitoring, and payment simulation.

## 🚀 Features

- **Authentication**: JWT with access & refresh tokens
- **Authorization**: Role-based (Admin, Instructor, Student)
- **Course Management**: Create, read, update, delete courses
- **Lesson Management**: Add lessons to courses
- **Enrollment**: Students can enroll in courses
- **Progress Tracking**: Track lesson completion percentage
- **Payments**: Simulated payment processing
- **Email Notifications**: Enrollment and payment confirmations
- **Advanced Features**: Pagination, filtering, sorting

## 📋 Prerequisites

- Python 3.8+
- MySQL 5.7+
- pip package manager
